## Installation
1. Clone the repo 
```sh
git clone https://github.com/SalmaaOsamaa/RenosystemsTask.git
```

2. install dependenscies
 ```bash
   npm install
  ```
   
3. run the project 
```bash
   npm start
   ```

<h6>In order to run db on localhost</h6>

2. install json
 ```bash
   npm install json-server
  ```
   
3. run this command
```bash
json-server --watch _db/db.json --port 4000
   ```
